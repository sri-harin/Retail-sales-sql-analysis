# 🛒 Retail Sales Analysis using SQL

## 📌 Project Overview

This project analyzes retail sales data to extract business insights such as customer behavior, product performance, and sales trends using SQL.

---

## 🎯 Objectives

* Analyze sales performance across categories
* Identify top customers and revenue contributors
* Understand time-based sales trends
* Perform data cleaning

---

## 🛠️ Tools Used

* MySQL
* SQL (Joins, Aggregations, Window Functions)
* CSV Dataset

---

## 📂 Project Structure

* `data/` → dataset
* `sql/` → SQL queries
* `outputs/` → result screenshots
